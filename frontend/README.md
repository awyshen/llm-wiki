# LLM Wiki 前端界面

## 项目概述
LLM Wiki 前端界面是一个现代化的知识库管理系统前端，使用 HTML、CSS 和 JavaScript 构建，提供直观的用户交互体验。

## 版本说明

### v1.0.0 (2026-04-20)

#### 主要功能变更
- **完整的模块结构**：实现了文档导入、文档管理、文档处理、Wiki 浏览、高级搜索、知识图谱和系统状态等功能模块
- **现代化界面设计**：采用响应式布局，支持不同屏幕尺寸
- **知识图谱可视化**：使用 D3.js 实现完整的 node-link-node 关系结构可视化
- **交互式操作**：支持节点拖拽、缩放、平移等交互功能
- **路径查询与相关实体**：实现了知识图谱的路径查询和相关实体查询功能
- **导出功能**：支持将知识图谱导出为 JSON 或 CSV 格式

#### 性能优化
- **图形渲染优化**：使用 D3.js 力导向布局，确保在图谱规模较大时仍保持良好性能
- **响应式设计**：适配不同设备屏幕，提供一致的用户体验
- **代码结构优化**：模块化设计，提高代码可维护性

#### 已知问题及修复情况
- **修复了 centerNode 未定义错误**：确保知识图谱相关实体查询功能正常运行
- **修复了路径查询功能**：解决了路径查询时的变量未定义问题
- **修复了图谱网络可视化问题**：确保图谱能够正常渲染和展示

## 技术栈
- **前端框架**：原生 HTML5, CSS3, JavaScript
- **可视化库**：D3.js v7
- **样式设计**：CSS 变量、Flexbox 布局
- **服务器**：Python HTTP Server

## 快速开始

### 启动前端服务器
```bash
cd frontend
python3 -m http.server 8080
```

### 访问界面
打开浏览器，访问 http://localhost:8080/

## 功能模块

### 1. 文档导入
- 支持批量导入多种格式的文档
- 显示导入结果和状态

### 2. 文档管理
- 查看文档列表和状态
- 按状态筛选文档
- 查看文档详细信息

### 3. 文档处理
- 处理待处理的文档
- 查看处理统计信息

### 4. Wiki 浏览
- 浏览知识库页面
- 按分类筛选页面
- 搜索页面内容

### 5. 高级搜索
- 支持语义搜索和模糊搜索
- 按分类和页面类型筛选
- 查看搜索历史

### 6. 知识图谱
- 可视化知识图谱结构
- 路径查询功能
- 相关实体查询
- 图谱导出功能

### 7. 系统状态
- 查看系统运行状态
- 监控系统性能
